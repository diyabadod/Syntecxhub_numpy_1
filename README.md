# Company Productivity Analyzer using NumPy

A Python project that demonstrates the use of **NumPy**, **Matplotlib**, and **Object-Oriented Programming (OOP)** to analyze employee productivity across multiple company branches. The application reads production data from a text file, performs statistical analysis, calculates employee salaries, and visualizes productivity using graphs.

# 📌 Project Overview

This project processes employee production data stored in a text file using NumPy. It identifies the most and least productive company branches, calculates average productivity, computes employee salaries based on production, and generates productivity graphs using Matplotlib.

The project was developed as part of my internship to strengthen my understanding of Python programming, data analysis, and visualization.

# 🚀 Features

- Read production data from a text file
- Store and process data using NumPy arrays
- Calculate total productivity for each company branch
- Identify the most productive branch
- Identify the least productive branch
- Calculate average productivity per employee
- Calculate the overall company average productivity
- Calculate employee salaries based on products manufactured
- Visualize employee productivity using line charts
- Demonstrate Object-Oriented Programming using a custom `IntArray` class
